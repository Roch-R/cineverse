Supports: YouTube, Google Drive (share link), Dropbox, OneDrive, Vimeo, or any direct video URL. This is the only way to post videos above ~1 GB.

📘 How to post huge videos (10 GB +)
  1. Google Drive (free, up to 15 GB): upload the file to Drive → right-click → Share → Anyone with the link → copy the link and paste into the URL tab.
  2. YouTube (free, unlimited): upload as Unlisted, copy any YouTube URL (watch / share / embed) — it'll auto-convert.
  3. Cloudflare R2 / Bunny.net / Backblaze B2 — cheap object storage. Get the direct .mp4 URL and paste it.
  4. Internet Archive (archive.org) — free permanent hosting for public content.

⚠ Browser-based uploads can never hold 10 GB — localStorage is 5 MB, IndexedDB is ~1 GB on most browsers. External hosting is the only real option.
